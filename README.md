### Hướng dẫn Cài đặt

* **Cài đặt các gói phụ thuộc (dependencies):**
    ```bash
    dotnet restore
    ```
* **Chạy dự án:**
    ```bash
    dotnet run
    ```
    *(Hoặc bạn có thể nhấn nút Run/Debug trong Visual Studio / VS Code)*
